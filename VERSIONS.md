# SuperClipboard 版本日志

## v3.0.0 (2026-07-09)

新版界面视觉升级与实验功能整理版本。

- 新版 UI 加入毛玻璃标题区、Tab 栏、设置菜单、确认弹窗、列表条目和底栏视觉
- 新版 UI 下备忘录条目新增左侧配色边框，使用备忘录独立配色
- 实验功能新增“隐藏条目色条”，可隐藏剪贴板条目左侧色条和原文左侧边框，默认关闭
- 优化新版 UI 的 Tab 半透明质感、间距、选中阴影和滚动层级
- 原格式预览和查看原文区域改用独立中性背景，不再受主题配色影响
- 设置、存储设置和实验功能菜单统一为毛玻璃浮层，并补充 WebView 兼容样式
- 清理开发运行日志目录 `.codex-run/` 和旧 `tauri-dev.log`
- 应用、Tauri 配置、Rust 包、文档和官网展示同步为 `3.0.0`

## v2.5.0 (2026-07-09)

界面字段对齐、实验功能入口与存储设置正式化版本。

- 统一备忘录底部时间、编辑时间、分类标签和“显示更多”的垂直居中
- 优化备忘录底部布局，“显示更多”不再挤占标签区域
- 统一剪贴板标签、回收站倒计时、分类 Tab 和新建备忘录按钮的字段居中规则
- 新增实验功能入口，实验功能默认关闭
- “新版界面”和“剪贴板多标签显示”迁入实验功能，默认关闭
- 存储设置入口改为默认显示，外部 PostgreSQL 存储不再标记为测试版
- 引入 lucide-react 并统一主要操作图标
- 备忘录自动标签复用剪贴板分类器，减少代码误判和旧自动标签残留
- 官网版本同步为 `1.5`，软件版本同步为 `2.5.0`

## v2.3.2 (2026-07-08)

交互细节与窗口拖动体验优化版本。

- 窗口拖动期间临时关闭界面过渡动画，并为列表条目增加渲染隔离
- 取消普通窗口聚焦触发数据刷新，改为休眠/网络恢复检测，减少拖动窗口时的卡顿
- 剪贴板分类标签支持直接拖拽排序，并新增“标签排序”开关
- 存储设置标题右侧显示“本地存储 / 外部存储”状态，移除重复状态行
- 应用、Tauri 配置、Rust 包、文档和官网版本同步为 `2.3.2`

## v2.3.0 (2026-07-08)

备份格式、外部模式显示保护与更新检查修复版本。

- 修正版本比较逻辑，将单数字 minor 版本按发布习惯归一化，避免版本线再次回退
- 备忘录编辑保存后立即刷新本地 `updated_at`，退出编辑态后马上显示“编辑于刚刚”
- 本地备份/恢复改为新版 `.scbackup` 包格式，内含清单、数据和校验信息
- 备份/恢复标题标记为测试版，并显示备份创建版本、当前恢复版本与跨版本恢复风险
- 存储设置切换到外部模式时隐藏备份/恢复区域，避免误以为已备份外部 PostgreSQL 数据
- 存储设置面板显示本地 / 外部连接状态，保存切换后立即刷新
- 剪贴板、备忘录和回收站搜索改为多关键词匹配
- 创建备份和恢复备份拆分进行中状态，按钮文案不再互相串联
- 新版不再读取旧 gzip `.scbackup` 和裸 `.json` 备份
- 窗口唤醒、剪贴板变化和存储模式切换统一走前端事件总线触发刷新
- 应用、Tauri 配置、Rust 包、文档和官网版本同步为 `2.3.0`

## v2.2.0 (2026-07-08)

存储设置测试版与许可文档整理版本。

- 系统设置新增 `存储设置(测试版)` 开关，默认隐藏存储配置入口

- 外部 PostgreSQL 模式保存时自动测试连接并初始化远端表，切换后实时刷新列表和统计
- 外部模式在电脑从休眠或网络恢复后自动刷新列表和标签数量
- 手动连接配置中账号与数据库输入框位置调整
- 原备份/恢复入口移动到存储设置内统一管理
- 官网加入 Google Search Console 验证标记，移除未支持平台说明
- 仓库新增非商用许可文件，文档明确不允许未经授权的商业用途
- 应用、Tauri 配置和 Rust 包版本同步为 `2.2.0`

## v2.1.0 (2026-07-03)

质量修复与技术整理版本。

- 修复图片剪贴板去重：改为使用真实像素内容计算 SHA-256，避免同尺寸图片被误判为重复
- 修复搜索防抖后的旧关键词查询和重复请求问题
- 剪贴板/备忘录查询参数增加范围修正，`LIMIT/OFFSET` 改为参数绑定
- 备忘录手动标签跟随备忘录配色，自动标签继续沿用剪贴板分类颜色
- 备忘录自动标签类型由后端分类器统一判断，减少前后端规则漂移
- 回收站备忘录样式改为主题变量，深色模式更一致
- 拆分确认弹窗和回收站备忘录条目组件，降低主界面文件复杂度
- 数据库写入 `schema_version`，为后续稳定迁移打基础

## v2.0.0 (2026-07-01)

### 新功能

- **回收站**：备忘录软删除归档，归档 Tab 分「剪贴板」和「备忘录」子标签，30 天自动清理，显示剩余天数
- **正式命名**：软件名/项目名统一为 `SuperClipboard`，中文名为「超级剪贴板」
- **仓库名同步**：GitHub 仓库名同步为 `SuperClipboard`
- **首次语言跟随系统**：首次启动自动按系统语言选择中英文，托盘菜单同步
- **备忘录图片粘贴**：编辑区支持 Ctrl+V 粘贴图片，预览区直接渲染
- **备忘录拖拽排序**：Pointer Events 实现，固定条目不参与拖拽
- **自定义备忘录颜色**：8 种预设色 + 自定义 HEX 输入，独立于主题色
- **剪贴板内容编辑**：编辑后显示最后编辑时间，原内容折叠保留
- **托盘重置位置**：右键托盘「设置」自动重置为默认位置
- **主题三段式切换**：浅色 / 深色 / 跟随系统

### 改进

- 「归档模式」更名为「回收站」，图标统一为废纸篓
- 分类算法升级为评分制，新增 Markdown、YAML/TOML 检测，减少误判
- 默认窗口位置改为右半屏居中，快捷键唤起保留上次位置
- 底栏按页面分别显示存储占用，高度统一
- 备忘录暗黑模式适配，标签颜色与剪贴板分类徽章一致
- 默认快捷键改为 Alt+X
- 从 `SuperClipboard3` 改名升级时自动迁移旧数据目录

### 修复

- 快捷键注册竞态、录制支持 Windows 键
- 单实例启动，修复重复点击快捷图标生成重复托盘图标
- 窗口位置保存状态与后端对齐，移动防抖
- 备忘录自定义颜色在樱花主题下不被覆盖
- 备忘录列表切换乱跳、hover 残留
- UTF-8 多字节字符截断崩溃
