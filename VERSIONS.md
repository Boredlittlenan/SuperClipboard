# SuperClipboard 版本日志

## v2.15.1 (2026-07-06)

新增更完整的更新检查体验。

- 检查更新结果显示当前版本、最新版本、Release 名称和更新日志摘要
- 保留下载按钮，发现新版本后可直接跳转 GitHub Release
- 备份/恢复入口暂时隐藏，等待更适合大量数据的 SQLite 或压缩备份格式

---

## v2.1.0 (2026-07-03)

质量修复与技术整理版本。

- 修复图片剪贴板去重：改为使用真实像素内容计算 SHA-256，避免同尺寸图片被误判为重复
- 修复搜索防抖后的旧关键词查询和重复请求问题
- 剪贴板/备忘录查询参数增加范围修正，`LIMIT/OFFSET` 改为参数绑定
- 备忘录手动标签跟随备忘录配色，自动标签继续沿用剪贴板分类颜色
- 备忘录自动标签类型由后端分类器统一判断，减少前后端规则漂移
- 回收站备忘录样式改为主题变量，深色模式更一致
- 拆分确认弹窗和回收站备忘录条目组件，降低主界面文件复杂度
- 数据库写入 `schema_version`，为后续稳定迁移打基础

### 后续技术路线

- 将备忘录图片从正文 data URL 逐步迁移到独立图片存储/缓存
- 为大量历史记录引入虚拟列表或分页加载
- 补充端到端测试覆盖快捷键、托盘、更新检查和备忘录图片排序

## v2.0.1 (2026-07-01)

小版本维护更新。

- 软件启动后直接显示桌面主窗口，不再只隐藏到托盘
- 包含单实例启动修复：反复点击桌面快捷图标或重复启动程序时会唤起已有窗口，不再创建多个进程和重复托盘图标
- 文档记录“点击即粘贴”现状：当前依赖隐藏窗口后原输入窗口恢复焦点，再延迟发送 Ctrl+V；部分软件或焦点切换较慢时可能不够灵敏，后续单独重构
- 版本号同步为 `2.0.1`

## v2.0.0 (2026-07-01)

### 新功能

- **回收站**：备忘录软删除归档，归档 Tab 分「剪贴板」和「备忘录」子标签，30 天自动清理，显示剩余天数
- **正式命名**：软件名/项目名统一为 `SuperClipboard`，中文名为「超级剪贴板」
- **仓库名同步**：GitHub 仓库名同步为 `SuperClipboard`
- **首次语言跟随系统**：首次启动自动按系统语言选择中英文，托盘菜单同步
- **备忘录图片粘贴**：编辑区支持 Ctrl+V 粘贴图片，预览区直接渲染
- **备忘录拖拽排序**：Pointer Events 实现，固定条目不参与拖拽
- **自定义备忘录颜色**：8 种预设色 + 自定义 HEX 输入，独立于主题色
- **剪贴板内容编辑**：编辑后显示最后编辑时间，原内容折叠保留
- **托盘重置位置**：右键托盘「设置」自动重置为默认位置
- **主题三段式切换**：浅色 / 深色 / 跟随系统

### 改进

- 「归档模式」更名为「回收站」，图标统一为废纸篓
- 分类算法升级为评分制，新增 Markdown、YAML/TOML 检测，减少误判
- 默认窗口位置改为右半屏居中，快捷键唤起保留上次位置
- 底栏按页面分别显示存储占用，高度统一
- 备忘录暗黑模式适配，标签颜色与剪贴板分类徽章一致
- 默认快捷键改为 Alt+X
- 从 `SuperClipboard3` 改名升级时自动迁移旧数据目录

### 修复

- 快捷键注册竞态、录制支持 Windows 键
- 单实例启动，修复重复点击快捷图标生成重复托盘图标
- 窗口位置保存状态与后端对齐，移动防抖
- 备忘录自定义颜色在樱花主题下不被覆盖
- 备忘录列表切换乱跳、hover 残留
- UTF-8 多字节字符截断崩溃

## v1.5.0 (2026-06-23)

- 标签栏支持滚轮滑动

### 修复
