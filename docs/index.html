Windows 剪贴板工作台

复制过的内容，不该只停留几秒。

SuperClipboard 把剪贴板历史、备忘录、回收站、主题和托盘快捷操作收进一个轻量窗口里。默认本地保存，也可以在存储设置中连接自己的 PostgreSQL。

核心能力

像工作台一样整理复制历史

高频复制、查找、备忘和恢复都在同一个窗口里完成，不把简单动作变成复杂流程。

Aa

智能分类

自动识别文本、链接、图片、代码、邮箱和文件路径，搜索时更快定位。

#

备忘录

标题、正文、图片预览、标签、置顶和拖拽排序，为临时信息留一个清爽位置。

{ }

编辑与原文

剪贴板条目可以编辑，原始内容可保留查看，备忘录默认使用格式化预览。

30

回收站

误删内容可恢复，剪贴板和备忘录分栏查看，并按剩余天数提示清理状态。

UI

主题细节

支持浅色、深色、跟随系统和蓝色/粉色强调色，备忘录配色保持独立。

↵

托盘和快捷键

托盘常驻、开机自启、全局快捷键和单实例唤起，适合每天反复打开。

数据与隐私

先本地可靠，再给外部存储留入口

默认数据保存在本机 SQLite。需要多设备方案时，可在存储设置中连接自己的 PostgreSQL 数据库。

默认模式

本地 SQLite

剪贴板、备忘录和设置默认写入本机应用数据目录，软件不会主动上传内容。

可选模式

外部 PostgreSQL

用户可以保存多个数据库连接并快速切换。外部模式下正文数据写入外部数据库。

工具

备份 / 恢复

备份入口已归入存储设置，本地模式使用新版 .scbackup 包并记录版本与校验信息。
